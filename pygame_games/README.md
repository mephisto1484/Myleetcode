# Pygame 简单小游戏合集

这里包含了多个使用 pygame 开发的简单小游戏，适合学习游戏开发基础。

## 安装依赖

首先确保已安装 pygame：

```bash
pip install pygame
```

## 游戏列表

### 经典游戏

#### 1. 贪吃蛇 (snake_game.py)

经典的贪吃蛇游戏。

**操作方式：**
- 使用方向键（↑↓←→）控制蛇的移动方向
- 吃到红色食物可以增加分数和长度
- 撞到自己或墙壁游戏结束
- 按空格键重新开始

**运行命令：**
```bash
python snake_game.py
```

#### 2. 打砖块 (breakout_game.py)

经典的打砖块游戏，消除所有砖块获胜。

**操作方式：**
- 使用左右方向键（←→）或 A/D 键移动挡板
- 用挡板反弹球来击碎砖块
- 消除所有砖块获胜
- 球掉落会失去生命，生命值归零游戏结束
- 按空格键重新开始

**运行命令：**
```bash
python breakout_game.py
```

#### 3. 接球游戏 (catch_game.py)

接住下落的彩色球，避免让球掉落。

**操作方式：**
- 使用鼠标左右移动控制挡板
- 接住下落的彩色球获得分数
- 有漂亮的粒子特效
- 游戏会随着分数增加而难度递增
- 失误5次游戏结束
- 按空格键重新开始

**运行命令：**
```bash
python catch_game.py
```

### 新增游戏

#### 4. 乒乓球 (pong_game.py)

单人对战AI的乒乓球游戏。

**操作方式：**
- 使用 W/S 键或 ↑/↓ 键控制左侧挡板
- 与AI对手对战
- 先得7分者获胜
- 难度会随着失分逐渐增加
- 按空格键重新开始

**运行命令：**
```bash
python pong_game.py
```

#### 5. 无尽奔跑 (runner_game.py)

无尽奔跑躲避障碍物游戏。

**操作方式：**
- 按空格键跳跃躲避障碍物
- 收集金币获得额外分数
- 游戏速度会逐渐加快
- 撞到障碍物游戏结束
- 按空格键重新开始

**运行命令：**
```bash
python runner_game.py
```

#### 6. 俄罗斯方块 (tetris_game.py)

经典俄罗斯方块游戏。

**操作方式：**
- ← → 键移动方块
- ↑ 键旋转方块
- ↓ 键加速下落
- 空格键直接落下
- 消除满行获得分数
- 按空格键重新开始（游戏结束时）

**运行命令：**
```bash
python tetris_game.py
```

#### 7. 台球 (billiards_game.py)

简易台球游戏，将所有球打入袋中获胜。

**操作方式：**
- 鼠标移动瞄准方向
- 在白球附近点击并按住鼠标左键
- 拖动鼠标设置击球力度（有力度条显示）
- 释放鼠标击球
- 目标：将所有15个目标球打入6个球袋
- 按 R 键重新开始

**运行命令：**
```bash
python billiards_game.py
```
